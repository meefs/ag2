</h2>

```python
WikipediaPageLoadTool(
    language: str = 'en',
    top_k: int = 3,
    truncate: int = 4000,
    verbose: bool = False
)
```

    A tool to load full Wikipedia page content along with metadata.<br/>This tool utilizes a language-specific Wikipedia client to search for relevant articles
    and returns a list of Document objects containing truncated page content along with rich
    metadata such as source URL, title, page ID, timestamp, word count, and size. It is ideal
    for agents requiring comprehensive, structured Wikipedia data for research, summarization,
    or contextual enrichment.<br/>Attributes: <br/>    language (str): The language code for the Wikipedia edition (default is "en" for English).<br/>    top_k (int): The maximum number of pages to retrieve per query (default is 3).<br/>    truncate (int): The maximum number of characters to include from each page's content (default is 4000).<br/>    verbose (bool): If True, enables verbose output for debugging purposes (default is False).<br/>    tool_name (str): The name of the tool, used in the User-Agent header.<br/>    wiki_cli (WikipediaClient): An instance of the WikipediaClient for interacting with the Wikipedia API.<br/>Initializes the WikipediaPageLoadTool with the specified parameters.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `language` | **Type:** str<br/><br/>**Default:** 'en' |
| `top_k` | **Type:** int<br/><br/>**Default:** 3 |
| `truncate` | **Type:** int<br/><br/>**Default:** 4000 |
| `verbose` | **Type:** bool<br/><br/>**Default:** False |
