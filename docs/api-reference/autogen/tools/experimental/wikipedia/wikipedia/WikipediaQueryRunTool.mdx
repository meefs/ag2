```python
WikipediaQueryRunTool(
    language: str = 'en',
    top_k: int = 3,
    verbose: bool = False
)
```

    A tool for executing Wikipedia queries and returning summarized page results.<br/>This tool requires the optional `wikipediaapi` package to be installed.<br/>Provides controlled access to Wikipedia content with configurable result limits.<br/>Attributes: <br/>    language (str): Wikipedia language edition to use (e.g., 'en', 'es')
        top_k (int): Maximum number of page summaries to return (capped at MAX_PAGE_RETRIEVE)
        verbose (bool): Flag to enable debug logging
        wiki_cli (WikipediaClient): Internal client for Wikipedia API interactions
    
    Initialize the Wikipedia query tool.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `language` | **Type:** str<br/><br/>**Default:** 'en' |
| `top_k` | **Type:** int<br/><br/>**Default:** 3 |
| `verbose` | **Type:** bool<br/><br/>**Default:** False |
